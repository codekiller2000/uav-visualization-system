# 无人机辅助物联网数据收集可视化系统

## 项目简介

基于 React 的无人机辅助物联网数据收集可视化系统，用于展示三大研究场景的仿真实验结果，特别是多目标优化问题的帕累托解集交互式选择功能。

## 功能特性

### 三大研究场景
1. **信息年龄优化** - 基于深度强化学习的最小化信息年龄优化场景
2. **能量效率优化** - 基于多目标优化的最大化数据价值和覆盖质量场景
3. **资源利用优化** - 动态环境下最大化资源利用效率优化场景

### 核心功能模块
- **用户管理** - 注册、登录、权限控制
- **场景配置** - 参数配置与实验提交
- **实验结果** - 单目标优化结果对比
- **帕累托决策** - 多目标优化解集展示与交互式选择

## 技术栈

- React 18.2
- React Router 6.20
- Vite 5.0
- 原生 CSS

## 核心功能说明

### 帕累托解集选择
在多目标优化场景中，系统会展示帕累托最优解集列表。用户可以：
1. 浏览所有帕累托解的关键指标（覆盖数量、数据价值）
2. 查看每个解的策略倾向（广覆盖/高价值/平衡）
3. 点击"选择此方案"查看完整的解决方案详情
4. 查看无人机访问顺序、能量消耗等详细信息

### 数据模拟
当前版本使用 Mock 数据模拟实验结果，便于演示和开发。后续可以对接真实的后端 API。
